# flutter_automation

## Initial Setup

- Install Flutter
- Install Android Studio
- Install Xcode
- Setup AVDs and Simulators

Execute **BDD** Tests

```sh
flutter driver --target=test_driver/bdd.dart
```

Execute **e2e** Tests

```sh
flutter driver --target=test_driver/e2e.dart
```
